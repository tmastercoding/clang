Korean ver:

문제
N개의 정수가 주어진다. 이때, 최솟값과 최댓값을 구하는 프로그램을 작성하시오.

입력
첫째 줄에 정수의 개수 N (1 ≤ N ≤ 1,000,000)이 주어진다. 둘째 줄에는 N개의 정수를 공백으로 구분해서 주어진다. 모든 정수는 -1,000,000보다 크거나 같고, 1,000,000보다 작거나 같은 정수이다.

출력
첫째 줄에 주어진 정수 N개의 최솟값과 최댓값을 공백으로 구분해 출력한다.

예제 입력 1 
5
20 10 35 30 7
예제 출력 1 
7 35

English ver:

problem

N integers are given. At this time, write a program to find the minimum and maximum values.

input
The first line gives the number of integers N (1 ≤ N ≤ 1,000,000). The second line contains N integers separated by spaces. All integers are greater than or equal to -1,000,000 and less than or equal to 1,000,000.

Print
Prints the minimum and maximum values ​​of the N integers given in the first line, separated by a space.

Example input 1 
5
20 10 35 30 7
Example output 1 
7 35

Python Solution:

# SOL 1
# get size
size = int(input())
# numbers = input().split(maxsplit=size)

# split into list
numbers = input().split(maxsplit=size)

# numbers_cpy = []
# for num in numbers:
#     numbers_cpy.append( int(num) )

# typecast to int
numbers_cpy = map( lambda x : int(x), numbers )

# print(numbers_cpy)

# get min and max
minVal = int(numbers[0])
maxVal = int(numbers[0])

for num in numbers_cpy:
    if num < minVal:
        minVal = num
    if num > maxVal:
        maxVal = num

# output
print(minVal, maxVal)

# SOL 2
import numpy as np

# get size
size = int(input())
# numbers = input().split(maxsplit=size)

# get input into list
numbers = input().split(maxsplit=size)

# create np array and typecast to int
numbers =  np.array( numbers ).astype(int)

# print min and max
print(numbers.min(), numbers.max())

C++ Solution:

#include <iostream>
#include <string>
using namespace std;

int main(){
    // declare variables
    int size = 0;
    int max, min, temp;
    int idx = 0;

    // input for size
    cin >> size;

    // restrictions for input

    // loop for size amount
    while(idx < size){
        // input temp
        cin >> temp;

        // assign temp to var
        // if first time
        if(!idx){
            max = temp;
            min = temp;
        } else {
            // if greater
            if(temp > max){
                max = temp;
            // if smaller
            } else if(temp < min){
                min = temp;
            }
        }
        idx++;
    }

    // output
    cout << min << ' ' << max << endl;
    return 0;
}
